# TrendGear Dashboard — De la Data Sintética a la Web Funcional


## Prompts utilizados con la IA

### 1. Priming

> "Hola ¿me podés ayudar a crear una página web? Te voy a explicar cómo la quiero."

### 2. Meta-prompting

> "¿Cómo te tengo que explicar lo que quiero para que quede bien? ¿Qué información necesitás?"

### 3. Prompt final de diseño

> "Quiero un dashboard para una tienda de tecnología que se llama TrendGear. Que tenga un menú arriba, una tabla con los clientes abajo, y un pie de página. Los colores: fondo negro,gris oscuro y los botones en azul. La letra que sea Roboto. Que el menú se convierta en un botón de hamburguesa cuando se ve desde el celular. Y necesito el código en 3 archivos separados: uno de HTML, uno de CSS y uno de JavaScript."

### 4. One-shot prompting

>"Te paso una tabla de ejemplo con 15 clientes. Necesito que hagas un script en Python que me genere 100 clientes más, siguiendo el mismo formato y que los datos tengan sentido (edades normales, fechas correctas, sin IDs repetidos)."
